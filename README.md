# X-Password
Um programa gerenciador de senhas para windows simples e intuitivo.

É necessário ter o Java JRE 9 ou superior para rodar o programa no seu computador
Download [Java JRE 9.0.4](https://www.filehorse.com/download-java-runtime-64/33592/)

É recomendável utilizar a versão mais recente do programa, versões anteriores podem conter bugs e falhas na segurança.
